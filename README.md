## Installation

eCrowJS's Theme can be installed as an [npm package](https://www.npmjs.com/package/@ecrowjs/theme).

```bash
# Install with NPM
npm install @ecrowjs/theme
# Install with Yarn
yarn add @ecrowjs/theme
```
